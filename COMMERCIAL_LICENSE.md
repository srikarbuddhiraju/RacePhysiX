## Who needs a commercial licence?

A commercial licence is required if any of the following apply:

| Scenario | Needs commercial licence? |
|---|---|
| Personal use, learning, research | ❌ No — AGPL-3.0 is free |
| Formula Student / FSAE team (non-commercial) | ❌ No — AGPL-3.0 covers this |
| Open-source project (source publicly available) | ❌ No — AGPL-3.0 covers this |
| University teaching / course materials | ❌ No — AGPL-3.0 covers this |
| Closed-source commercial product embedding RacePhysiX | ✅ Yes |
| White-label / rebranded deployment | ✅ Yes |
| SaaS service built on RacePhysiX without releasing source | ✅ Yes |
| Commercial motorsport consultancy using it as a client tool | ✅ Yes |
| Reselling RacePhysiX or a derivative product | ✅ Yes |

---

## What a commercial licence includes

- Right to use RacePhysiX in closed-source commercial products
- Right to run as a hosted service without AGPL source disclosure obligation
- Right to white-label or rebrand
- No requirement to open-source your modifications
- Attribution requirement: "Powered by RacePhysiX" visible to end users
